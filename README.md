#  NETFLIX CONTENT ANALYSIS DASHBOARD | TABLEAU 

##  Project Overview

This project explores the Netflix Titles dataset using Tableau to build a concise one-page analytical dashboard.
The dashboard provides a clear visual summary of:
- Distribution of Movies vs TV Shows  
- Global content availability  
- Content growth over time  
- Genre distribution  
- Audience maturity ratings  
The focus of this project is clean layout design, visual storytelling, and structured data interpretation using Tableau.
---
##  Dataset
**Dataset:** Netflix Titles Dataset  
**Domain:** Entertainment / Media Analytics  
Each row in the dataset represents one Netflix title, which can be either a Movie or a TV Show.
Key fields used in this project:
- `title`
- `type` (Movie / TV Show)
- `country`
- `release_year`
- `rating`
- `listed_in` (Genre)
- `duration`

##  Dashboard Overview
![image alt](https://github.com/shailygupta-05/netflix-content-analysis-tableau/blob/7a04c9269e39bb1a6ffba8e11194823281654302/image/Untitled.png)
The one-page Tableau dashboard includes the following visualizations:
- 1 Movies vs TV Shows Distribution  - Displays the percentage split between Movies and TV Shows available on Netflix.
- 2️ Content by Country  - A map visualization showing global distribution of Netflix content.
- 3️ Content Growth Over Years  - An area chart highlighting the increase in Netflix content over time.
- 4️ Top 10 Genres  - Shows the most dominant content categories available on Netflix.
 -5️ Ratings Distribution  - Displays content classification based on audience maturity levels.
  
**Note:**  
**TV-MA (Television – Mature Audience)** indicates content intended for adult viewers (17+), often including strong language, violence, or mature themes.

---
##  Key Insights

- Netflix catalog contains a higher proportion of Movies compared to TV Shows.
- A majority of titles fall under the TV-MA rating, indicating a strong focus on mature audiences.
- The United States contributes the highest number of Netflix titles.
- Content production increased significantly after 2015.
- Documentaries and Stand-Up Comedy are among the most frequent genres.
